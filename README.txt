Φάκελος Results:
	-> Γραφήματα
	-> Αρχεία .txt με τις μετρήσεις
	-> graph_benchmark.py python script που δημιουργεί τα γραφήματα

Anafora.pdf:
	-> Η αναφορά που μας.

benchmark.c:
	Κώδικας c που χρησιμοποιήσαμε προκειμένου να αυτοματοποιήσουμε την διαδικασία εκτέλεσης και χρονομέτρησης των παράλληλων προγραμμάτων.
Με ./benchmark μπορείτε να δείτε τα option για την εκτέλεσή του.

multistart_hooke_omp.c:
	Παράλληλος c κώδικας σε ΟpenMP (1ο ερώτημα)

multistart_hooke_omp_tasks.c:
	Παράλληλος c κώδικας σε ΟpenMP - tasks (2ο ερώτημα)

multistart_hooke_mpi.c:
	Παράλληλος c κώδικας σε MPI (3ο ερώτημα)

multistart_hooke_mpi_omp.c:
	Παράλληλος c κώδικας σε MPI - ΟpenMP (4ο ερώτημα)

Makefile:
	Το ενημερωμένο με όλα τα αρχεία .c makefile που μας δόθηκε
